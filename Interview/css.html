# CSS垂直居中

<div class="box">
  <div class="small">small</div>
</div>
1
2
3
  • 使用子绝父相，定位垂直50%，然后使用margin或者calc减去高度的一半。
.box {
  position: relative;
  width: 300px;
  height: 300px;
  background-color: red;
}
.small {
  position: absolute;
  /* 
   * top: 50%;
   * margin-top: -50px; 
  */
  top: calc(50% - 50px);
  width: 100px;
  height: 100px;
  background-color: yellow;
}

# CSS三角形

宽高定位0，给边框宽度、颜色。

  • 给三角形对角一边的宽度设为0。
  • 给三角形对角一边的颜色和两侧的颜色设为透明transparent。
<style>
  .border {
    width: 0;
    height: 0;
    border-style:solid;
    border-width: 0 50px 50px;
    border-color: transparent transparent #d9534f;
  }
</style>
<div class="border"></div>

# 左边固定，右侧自适应

  • 左侧float或者position脱离文档流，右侧使用margin-left隔开距离。
.box {
  overflow: hidden; /* 添加BFC，形成独立块级作用域，不收外部元素影响 */
}
.left {
  float: left;
  width: 200px;
  height: 400px;
  background-color: gray;
}
.right {
  margin-left: 210px;
  height: 200px;
  background-color: lightgray;
}

# 两边固定，中间自适应

  • 左右使用position定位上去，中间的使用margin隔开
.container {
  position: relative;
}
.left, .right {
  position: absolute;
  top: 0;
  width: 400px;
  height: 200px;
  background-color: green;
}
.left {
  left: 0;
}
.right {
  right: 0;
}
.center {
  height: 200px;
  margin: 0 400px;
  background-color: yellow;
}
